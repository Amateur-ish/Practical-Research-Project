HTML Basic Structure

There are several types of HTML Tags, but let's start of with the basics!

Paragraph Tag

The Paragraph tag <p> is used to create paragraphs. This is where you would want to put your sentences.

The Heading Tag

The Heading tag is defined as <h1> to <h6> is used to create headings on an HTML Page.
Usually use for headings such as subjects, title of a section, etc.

HTML Attributes

Let's say you want to show how to use a toy you're gifting someone. Instructions such as "Push the red button to make the toy move!" are similar to HTML attributes, which are additional characteristics about the items on a webpage.

HTML Attributes Example


In HTML, attributes provide the element more details and help control its behavior or appearance. Attributes are always specified in the opening tag of an element and come in name-value pairs.
